# GapStatistic.jl

Implements the [Gap statistic](https://web.stanford.edu/~hastie/Papers/gap.pdf) method for cluster count determination

## Example

X = cat(randn(20,2),randn(20,2).+5,dims=1)
k,g = GapStatistics.GapStatistic(X,samples=100)
p1 = UnicodePlots.lineplot(g)
p2 = UnicodePlots.scatterplot(X[:,1],X[:,2])
print("Number of Clusters: $k\n",p1,"\n",p2)
